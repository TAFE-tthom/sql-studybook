# SQL Study Book

Welcome to the SQL studybook, this book will focus on using SQL, primarily from a **MariaDB** perspective but will attempt to be vendor agnostic. This is a study book designed to complement a course and **SQLAttack** challenge website.

## Who is it for and why?

You are wanting to learn about databases, specifically relational databases. Your current excursion has landed you here. The goal of this book is to go through the fundamentals with databases and sql.

## System Requirements

As part of this book, you should have access to a computer that you can install **MariaDB**, **PostgreSQL**, **MS-SQL** and **SQLite**. **MariaDB** and **SQLite** are database software that are easy to install.

Please be aware of the following things:

* What operating system you currently running, you could be using
  * Windows
  * MacOS
  * Linux
  * FreeBSD
  * Something else entirely!

* Hardware Architecture of your device, while it likely won't make a meaningful difference, being aware of it when setting up will help. You could be using
  * x86 (x86_64, AMD64)
  * ARM (aarch64)


* Access to a terminal/shell to execute code and commands. you could be using
  * zsh
  * bash
  * powershell

* Access to a web browser to execute code, you could be using
  * Firefox
  * Chrome
  * Safari

After reviewing the above and seeing if you meet at least one of each category, you can start.
