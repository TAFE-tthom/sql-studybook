# Exercises


## Tasks

### 1. Wasted procurement

Write a query that returns all products that have never been ordered.

Use the tables from the Northwind Database: Products and OrderDetails


### 2. Busy day

Write a query that returns all orders placed on the first day of activity that can be found in the orders table.

Use the tables from the Northwind Database: Orders


### 3. Drinks list

Write a query that returns the products that have been ordered and belong to the 'Beverages' category


### 4. Wednesday is the day

Write a query that returns the products that were part of orders placed on a Wednesday.

Hint: You will need to use the `DAYOFWEEK` function, which takes a date and returns an integer value indicating the day. Please refer to the MySQL reference.


### 5. Last Action

Write a query that returns for each customer, all orders placed on the customer's last day of activity.

Hint: Remember that the subquery will be evaluated for each order record
  Tables you should consider: Orders
