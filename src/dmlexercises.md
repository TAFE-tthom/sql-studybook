# Exercises

## Tasks

### 1. Retrieving a list of customers

Prepare a list of customers ordered by country region and city.

Use the Northwind database and Customers table


### 2. Employees in the company

How many employees are there in the company?
Use the Northwind database and Employees table

Consider using the `COUNT` built-in function.


### 3. Finding the sauce!

List the products where the product name contains the word 'sauce'
Use the Northwind database and Products table

Consider using `'%word%'` for finding anything that contains `word`. Replace `word` with what your search term is.


### 4. Find all products that belong to an order (Multiple Tables)

Write a query that will list all products for a specific order ID.

Use the Northwind database and the OrderDetails and Products tables

You can specify multiple tables after the `FROM` keyword, make sure they are comma separated.
