# Review

In conjunction with functions, joins and sets, it is only normaly to start using aggregate functions to produce appropriate summaries or reductions of data.

* What you know and understand
  * Aggregate functions usage and context
  * Distinct and In keywords and where they are appropriate
  * Subquery construction and evaluation order

* What you should be able to apply
  * Be able to construct an aggregate function where appropriate
  * Using aggregate functions for performing numeric calculation and summation
  * Using a subquery to discover and extract data from a set to use in another
  * Applying predicates within an sql query on particular groups.
