# Review

* What you should know and understand

  * Difference between GRANT, DENY and REVOKE
  * What objects that permissions can be applied to
  * Difference and knowledge of users and roles

* What you should be able to apply
  * Creation of a user and role
  * Granting permissions onto a user and/or role
  * Revoking permissions onto a user and/or role
  * Denying permissions onto a user and/or role
