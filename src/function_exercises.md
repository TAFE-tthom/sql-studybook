# Exercises

## Tasks

### 1. Outputting Fullnames

You have been tasked with creating a query that will output all actors first name and last name in a single column. You will need to use the CONCAT function or || operator to merge the two columns into one.


### 2. Outputting Length of Names

You have been tasked finding out the length of each the last name for each actor. You will need to output their last name and the length of their last name.

You will also need to order by descending order.


### 3. Capitalisation

You have been tasked with otuputting the list of names but with capitalisation. You will need to use a mix of functions to achieve this.

Consider the following functions:

    SUBSTRING
    LOWER
    UPPER
    CONCAT

Do note, you can take the value that would be returned by the function as an argument for another.
